## 📂 Featured Projects

### 💧 Daily Water Intake Prediction
> Binary classification pipeline to predict hydration status using structured health data.
- Trained & compared **Logistic Regression**, **Random Forest**, and **ANN (MLPClassifier)**
- Achieved **99.8% accuracy** and **0.999 ROC-AUC**
- Applied feature engineering, standardization, and weighted F1-score evaluation

### 🩺 Breast Cancer Prediction
> Classification models to detect malignant vs. benign tumours using the Wisconsin Breast Cancer dataset.
- Implemented **Logistic Regression**, **Random Forest**, and a **TensorFlow/Keras neural network**
- Achieved **~94–96% accuracy**
- Evaluated using precision, recall, and F1-score

### 🥗 Nutrition Tracker
> Python-based nutrition tracking web app built with Flask.
- Features user authentication, session handling, meal logging, and macro tracking
- Custom food entry with structured Python data processing
- Version controlled with **Git/GitHub**
